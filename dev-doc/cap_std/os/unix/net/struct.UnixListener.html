pub struct UnixListener { /* private fields */ }
Expand description

A structure representing a Unix domain socket server.

This corresponds to std::os::unix::net::UnixListener.

This UnixListener has no bind method. To bind it to a socket address, first obtain a Dir containing the path, and then call Dir::bind_unix_listener.
