pub struct OwnedWriteable(_);
Expand description

An owning I/O handle that implements Write.

This doesn‚Äôt implement Into* or From* traits.

Platform-specific behavior

On Posix-ish platforms, this writes to the handle as if it were a [File]. On Windows, this writes to a file-like handle as if it were a [File], and to a socket-like handle as if it were a TcpStream.
